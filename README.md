📊 Exercícios em Python

Este repositório contém uma série de atividades desenvolvidas em Python com o objetivo de praticar lógica de programação, estruturas de repetição, condicionais e entrada de dados.

🚀 Atividades desenvolvidas

🗳️ Sistema de votação:

Programa que simula uma eleição com:

4 candidatos,
Votos nulos e em branco,
Validação de entrada de dados,
Contagem total de votos e
Cálculo de porcentagem de votos para cada categoria.

💡 Conceitos utilizados:

for e while,
Estruturas condicionais (if, elif, else),
Validação de entrada e
Operações matemáticas.

🎂 Classificação por faixa etária:

Programa que:

Solicita a idade do usuário,
Classifica em intervalos:
0 a 25 anos,
26 a 50 anos,
51 a 75 anos,
76 a 100 anos.
Encerra caso a idade seja inválida.

💡 Conceitos utilizados:

Loop infinito (while True)
Condições compostas
Controle de fluxo com break

🔢 Tabuada:

Programa que:

Recebe um número inteiro e
Exibe a tabuada de 1 a 10.

💡 Conceitos utilizados:

Laço for,
Operações matemáticas e
Formatação de saída.

🔍 Verificador de número primo:

Programa que:

Verifica se um número é primo e
Utiliza divisão para testar se há divisores além de 1 e ele mesmo.

💡 Conceitos utilizados:

Estrutura de repetição,
Operador módulo (%) e
Lógica booleana.

🧠 Objetivo

Essas atividades foram desenvolvidas com foco em:

Fortalecer a lógica de programação, praticar estruturas fundamentais do Python e desenvolver soluções simples e eficientes.

🛠️ Tecnologias utilizadas:

Python 3
